# `pontil_context` Changelog

## v1.0.0 / 2026-04-29

Initial release of `pontil_context`, adapted from [`@actions/github`][github] to
provide the `github.context` (`context.Context`) and `github.context.payload`
(the output of `context.event()`).

[github]: https://github.com/actions/toolkit/tree/main/core
